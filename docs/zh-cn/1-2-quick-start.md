# 快速开始

## 环境要求

- Python 3.10 或更高版本
- Git（用于克隆项目）

## 安装

### 方式一：从源码安装

```bash
# 克隆项目
git clone https://github.com/771835/dovetail.git
cd dovetail
 
# 安装依赖
pip install -r requirements.txt
 
# 验证安装
python main.py --help
```

### 方式二：下载预编译版本

1. 访问[Releases 页面](https://github.com/771835/dovetail/releases),下载最新版本。
2. 解压并找到主程序(通常为dovetail.exe)
