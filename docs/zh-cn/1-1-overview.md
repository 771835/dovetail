# Dovetail 概述

## 什么是 Dovetail？

Dovetail 是一个专为 Minecraft 数据包开发设计的编程语言和编译器工具链。它提供了面向对象的语法结构，让你能够用更现代、更易维护的方式编写
Minecraft 指令逻辑。

### 核心特性

- **面向对象编程**: 支持类、接口、继承等现代编程概念
- **类型安全**: 编译时类型检查，减少运行时错误
- **模块化设计**: 完整的包含系统和模块管理
- **多后端支持**: 通过插件架构支持不同 Minecraft 版本
- **优化编译**: 多级优化确保生成高效的指令代码

### 设计目标

1. **降低学习曲线**: 类似现代编程语言的语法
2. **提高代码复用**: 面向对象和模块化支持
3. **保证类型安全**: 编译时错误检测
4. **优化性能**: 生成高效的 Minecraft 指令

## 项目架构

```
 dovetail/
├── transpiler/ # 核心编译器
│ ├── core/ # 编译器核心组件
│ ├── plugins/ # 后端插件系统
│ └── utils/ # 工具函数
├── antlr/ # 语法解析器定义
├── example/ # 示例代码
├── docs/ # 文档
└── proposals/ # 语言提案
```

## 编译流程

1. **词法/语法分析**: 使用 ANTLR4 解析源代码
2. **AST 构建**: 生成抽象语法树
3. **语义分析**: 类型检查、符号解析
4. **IR 生成**: 生成中间表示指令
5. **优化处理**: 多级优化策略
6. **代码生成**: 通过后端插件生成目标指令
